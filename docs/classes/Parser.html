Class ParserInternal

Contains the DC file parser to process Distributed Class definitions.

Remarks

This parser was heavily inspired by an attempt to parse the DC file by bobbybee using JavaScript which was used in the Astron 'Web Panel' project, although abandoned in 2014. https://github.com/Astron/Web-Panel/blob/master/js/dcparser.js
