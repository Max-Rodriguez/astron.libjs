Class InternalRepository

Represents an internal connection to the Astron cluster for AI / UberDOG processes.

Param

Specify the DC filename, callbacks, AI channel, and other roles' channels.

Param

The IPv4 address of the Astron server. Defaulted to loopback address.

Param

The port to use in the connection. Defaulted to 7199.
